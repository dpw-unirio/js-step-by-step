Agora é hora de conhecer outros dois incríveis acompanhantes, primeiramente o AJAX!

AJAX é uma sigla para Asynchronous JavaScript and XML, ele é usado para fazer requisições (ou seja, carregar um arquivo ou fazer uma chamada ao seu servidor) assíncronas (ou seja, que não interrompem o carregamento ou uso da página).

Além dele, vamos descobrir o JSON (Javascript Object Notation), ele permite representar objetos do javascript em texto plano e provê uma forma conveniente para armazenar informações.

    • Para transformar a string retornada em um objeto você pode usar o método JSON.parse(text)
