Isso provavelmente vai ser um dever de casa!

Adicione uma seção aonde o usuário deva distribuir 20 pontos entre 5 atributos:

  • STR
  • DEX
  • INT
  • SPD
  • LUK

Tarefas:

  • Criar uma seção para obter os valores dos atributos.
    • Garantir que a soma deles não ultrapasse 20.
  • Salvar esses atributos no objeto ao salvar.
  • SIDE QUEST: Faça uma interface bonita!
    • Mostre a quantidade de pontos restantes pra distribuir.
      • Você pode usar o evento change para atualizar ele quando o valor de um dos campos mudar.
    • Adicione botões de + e - dos lados do campo.
    • Desabilite os botões de + quando a soma dos pontos for 20.
